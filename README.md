# eBarrios 🏡
eBarrios es un Sistema de Gestión Operativa pensado para barrios cerrados. La propuesta se centra en centralizar las tareas administrativas, de seguridad y de mantenimiento para que los residentes y la administración puedan gestionar todo desde un solo lugar sin depender de registros manuales.

Objetivo del proyecto
El objetivo principal de esta etapa es estructurar y validar el flujo de gestión del barrio:
Registro de residentes y viviendas
→ Control de accesos (visitantes y personal)
→ Carga y seguimiento de reclamos
→ Gestión de incidentes y tareas de mantenimiento
