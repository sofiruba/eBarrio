# eBarrio - Integrante 1

Módulo desarrollado: Residentes, Viviendas, Visitantes + JavaFX inicial.

## Contenido

- `Barrio.java`
- `Vivienda.java`
- `Residente.java`
- `Visitante.java`
- `MainApp.java`
- `styles.css`
- `texto_informe_integrante1.txt`

## Cómo ejecutar

Este proyecto está preparado para JavaFX. Se puede abrir en IntelliJ IDEA, Eclipse o VS Code con JavaFX configurado.

Clase principal:

```text
com.ebarrio.app.MainApp
```

La interfaz incluye una pantalla inicial de eBarrio con estética verde oscuro y una sección básica para visualizar residentes y visitantes.

## Nota

El método `crearReclamo()` queda preparado en `Residente`, pero sin implementación completa porque depende del módulo de solicitudes que desarrollan otros integrantes.
